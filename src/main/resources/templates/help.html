Pagina di aiuto per utilizzare al meglio le API offerte da quest'app

/api/assets/allsrcIPv4

indirizzo delle API per mostrare tutti i srcIPv4 degli assets memorizzati nel sistema. I dati vengono mostrati in formato JSON

/api/assets/alldestIPv4

indirizzo delle API per mostrare tutti i destIPv4 degli assets memorizzati nel sistema. I dati vengono mostrati in formato JSON

/api/assets/allsrcMAC

indirizzo delle API per mostrare tutti i srcMAC degli assets memorizzati nel sistema. I dati vengono mostrati in formato JSON

/api/assets/alldestMAC

indirizzo delle API per mostrare tutti i destMAC degli assets memorizzati nel sistema. I dati vengono mostrati in formato JSON

/api/assets?p=X&o=Y

indirizzo delle API per mostrare tutti i dati degli assets memorizzati nel sistema. I dati vengono mostrati in formato JSON.

Sono richiesti due parametri:

p -> indica la pagina da mostrare (i dati sono paginati 30 alla volta), X rappresenta un numero >= 0

o -> indica il parametro di ordinamento, Y rappresenta un numero da 1 a 5 (1 -> "DestIp4", 2 -> "SrcMac", 3 -> "DestMac", 4 -> "SrcIp6", 5 -> "DestIp6")
